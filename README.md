# /我的操作系统

> 像填 object 一样认识自己。

一个 Claude Code skill，生成交互式 HTML 自我复盘工具。支持两种模式：直接拿空白模板自填，或者让 Claude 采访你后生成有内容的个性化版本。

---

## 效果预览

五个 Tab，覆盖自我认知的完整链路：

| Tab | 内容 |
|-----|------|
| ① 变量数据 | CONST（环境/天赋/运气）+ VAR（价值观/思维方式/努力），用 object 结构整理 |
| ② 逻辑链 | IF→THEN、正向循环、矛盾冲突，写出变量之间的因果关系 |
| ③ 引导提问 | 6个模块 × 5道苏格拉底式问题，帮你从记忆里挖出真实答案 |
| ④ 人生坐标轴 | 把人生关键事件钉到时间轴上，按影响程度正负排布，SVG 可视化 |
| ⑤ 核心叙事 | 我是谁 / 我从哪里来 / 我为什么这样行动 → 合成一段"操作系统说明书" |

所有内容浏览器内直接编辑，数据自动保存到 localStorage，刷新不丢失。

## 安装

需要先安装 [Claude Code](https://claude.ai/code)。

```bash
git clone https://github.com/yueouLi/self-os-skill.git \
  ~/.claude/skills/我的操作系统
```

---

## 使用

在 Claude Code 里输入：

```
/我的操作系统
```

Claude 会问你选哪种模式：

**空白模板** — 5 秒拿到工具，自己在浏览器里填。  
**Claude 引导复盘** — 6 轮采访（约 20-30 分钟），Claude 根据你的回答生成预填好内容的个性化版本，并自动提炼逻辑链。

生成的 HTML 文件保存到桌面，浏览器打开即用，无需网络，无需账号。

---

## 背后的框架

把自我认知拆成两类变量：

```
CONST  不可控的初始值   →  环境 / 天赋 / 运气
VAR    可以主动调整的   →  价值观 / 思维方式 / 努力
```

每个变量记录：**表现[]**（行为上怎么体现）、**导致[]**（带来什么结果）、**条件**（什么前提下成立）、**依赖**（从哪里来）。

逻辑链把变量串起来，写出因果推导：

```
IF   环境.放手型父母 initialize() 完成
AND  成长中遭遇打压
THEN 批判性思维反向激活
∴    打压越强，主体性越强
```
