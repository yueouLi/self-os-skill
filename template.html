复盘方法：每道题先读一遍，闭眼想5秒，把第一个浮现的记忆/场景写下来。不用写得好，写得真才有效。 写完一个模块，回到「① 变量数据」Tab，把洞察填进去。
CONST

环境 — 18岁前，发生了什么？

▶
你的父母最常对你说的一句话是什么？那句话让你相信了什么？
18岁之前，谁是你最想模仿的人？谁是你最不想成为的人？为什么？
有没有一件"不公平"的事让你愤怒，但你当时什么也没做？那件事改变了你什么？
你小时候被允许的和被禁止的，各是什么？这塑造了你今天的哪个习惯？
如果用一个词描述你原生家庭的"氛围"，是什么词？那个氛围现在还在你身上吗？
CONST

天赋 — 天花板在哪里？

▶
回忆一件你不努力就做得很好的事，旁边的人需要很拼才能做到的。
什么事让你进入心流状态（忘了时间、不觉得累）？
你在哪件事上试了所有方法，都没法突破到顶尖水平？（这可能是真正的天花板）
别人经常找你帮什么忙？（别人公认的你的能力，往往是你自己最忽视的）
你最不喜欢但最容易完成的事是什么？（天赋不一定等于热爱）
CONST

运气 — 哪些事不在你控制里？

▶
你的一生中，有没有一个"如果没有那件事/那个人，我的路会完全不同"的时刻？
你哪次失败，后来证明是最好的事？失败带给你什么是成功带不来的？
你的出生/成长环境中，哪些是别人没有、你无意间得到的优势？（语言、城市、家庭开放程度）
有没有一个"偶遇"彻底改变了你的认知或方向？（陌生人、一本书、一句话）
VAR

价值观 — 你真正在乎什么？

▶
上一次你真的愤怒，是因为哪个原则被触犯了？
如果必须只做一件事，且只能以这件事评价你的一生，你会选什么？
哪件事你明知道"不聪明"也要做，因为它符合你的内心？
你最看不起什么样的人？（这往往反映你最珍视的东西）
你会为了什么主动降低自己的利益？（这里藏着你真正的优先级）
VAR

思维方式 — 你怎么处理信息和做决定？

▶
你最近一次改变主意是因为什么？是数据、情绪、还是某个人的话？
遇到别人批评你，你的第一反应是什么？先怀疑对方，先怀疑自己，还是搁置判断？
你解决一个新问题的第一步通常是什么？（查资料、找人问、自己推理、先动手试）
你有没有一个"不管别人怎么说，这件事我有自己的判断"的领域？那个判断从哪里来？
你做决定时，最难接受的是什么？（不确定性、失去控制、让别人失望、错失机会）
VAR

努力 — 你在做什么，为什么做？

▶
你现在最想做却一直没做的事是什么？是什么在阻止你？（时间、金钱、恐惧、还是你自己都不确定？）
你最近三个月，花时间最多的事是什么？这是你想要的时间分配吗？
你有没有一件事坚持了很久最后放弃了？事后判断：是"方向错了"还是"方法不对"？
你一天中最有精力的时间段是什么时候？你把什么放在那个时间段？
什么事让你越做越有劲（而不是越做越累）？那件事和你的"努力方向"重合吗？
用法：把人生中影响最大的时间节点钉到坐标轴上。正向事件（给你力量/机会/成长）往上，负向事件（打击/损失/错误）往下。 钉完再回到「① 变量数据」，你会发现每个 CONST/VAR 背后都有具体事件支撑。
